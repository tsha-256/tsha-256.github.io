Overview

Escape Mate is an interactive media project submitted to MHacks 2024. The project won the Interactive Media and Gaming track. Escape Mate is a platform to generate new escape room scenarios given a set of puzzle hardware components.

https://github.com/wenbox360/Escape_Mate

- Winner — Interactive Media & Gaming Track, MHacks 2024
